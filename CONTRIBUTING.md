# Contributing

This repository is a **read-only public example** maintained by Highnote.
It demonstrates a consumer prepaid card workflow using the
[`@highnote-oss/nodejs-sdk`](https://github.com/highnote-oss/nodejs-sdk)
and is not open to external contributions.

- **Found a bug or unclear example?** Contact your Highnote account team or
  support channel.
- **Found a security issue?** Please report it via GitHub's
  [private vulnerability reporting form](https://github.com/highnote-oss/highnote-consumer-prepaid-node/security/advisories/new) —
  see [`SECURITY.md`](SECURITY.md).
- **Looking for other examples or scenarios?** See the
  [`highnote-examples`](https://github.com/highnote-oss/highnote-examples)
  catalog.
- **External pull requests are auto-closed** by a workflow. We will still
  see any feedback you leave on the closed PR.

The rest of this document is for Highnote maintainers updating the example.

## Updating the example

Changes are PR-only. Direct pushes to `main` are disabled.

1. Open a branch from the latest `main`
2. Make the change, run the local checks (see `README.md`)
3. Open a PR; squash-merge once approved
